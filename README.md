# 坦克大战 (Tank Battle)

经典 FC 坦克大战的 HTML5 网页版实现。

## 🎮 游戏玩法

- **移动**: W A S D 或 ↑ ← ↓ → 方向键
- **射击**: 空格键 或 J 键
- **暂停**: P 键
- **开始/重新开始**: 空格键 或 Enter

## 🎯 游戏目标

保护基地（老鹰标志），消灭所有敌方坦克即可过关。

## ✨ 特色功能

- 🗺️ 多个关卡地图（3个预设 + 随机生成）
- 🧱 可破坏的砖墙和不可破坏的钢铁墙
- 💧 水域地形
- ⭐ 多种道具系统：
  - ⭐ 星星：升级坦克（速度、射速、双发、穿甲）
  - 💣 炸弹：清除所有敌人
  - ❤️ 生命：增加一条命
  - ❄️ 冰冻：暂停敌人移动
  - 🛡 护盾：短暂无敌
- 🤖 AI 敌人（随机移动 + 追踪玩家/基地）
- 📱 移动端触屏控制
- 🔊 音效反馈

## 🚀 运行方式

直接在浏览器中打开 `index.html` 即可游玩。

或使用任意 HTTP 服务器：

```bash
python3 -m http.server 8000
# 访问 http://localhost:8000
```
